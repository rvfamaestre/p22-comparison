\subsection{Motivation}
A fundamental trade-off exists in platoon control architectures:
\begin{itemize}
    \item \textbf{Local Control:} Relies on local sensors (radar/LiDAR). It is robust to network failures and ensures collision avoidance but is prone to string instability in long platoons.
    \item \textbf{Global Control:} Relies on V2V data from the leader. It provides excellent string stability but is highly sensitive to communication delays and packet loss.
\end{itemize}
